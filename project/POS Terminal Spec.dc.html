POS Terminal — Design Spec

Direction B · Dense Data-Forward, committed. Coral-red accent, Space Grotesk throughout, soft elevation, xl–2xl radii. Two genuinely different layouts — a counter Tablet (zero-scroll shell) and a handheld Phone (collapsible cart) — sharing one token system. Annotated screens, states, and an add-to-cart storyboard for engineering to build against.

Tablet ≥768px · landscape Phone ≤480px · portrait Accent #FF5533 GST 2.5% + 2.5%
